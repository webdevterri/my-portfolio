ABOUT ME

I learned to code in HTML in the late 90s - early 00s making personal sites on Geocities and Angelfire during the web 1.0 era on a clunky PC. I was so excited about building sites that I would think about it all day during school and practically run home, just down the block, so I can hop on the computer and build sites for hours on end!

That love of designing and coding sites never went away. All these years later, I'm still making sites and have finally delved further into prgramming learning Python and JavaScript. I have so many ideas there won't be enough time in the rest of my life to build it all but I want to do as much as I can.

I wrote an article explaining my journey learning JavaScript via Python (because my first attempt at JavaScript was a bust!) Ruby on Rails is a framework I'm very interested in picking up eventually as well. As you can see, I am a lifelong learner full of ideas.

I have a BA in Communications with a minor in Philosophy. I'm currently doing some Graphic Design work while pursuing a full-time web development career.
